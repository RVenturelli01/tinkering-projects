# Amateur Projects

A collection of personal projects built for fun during high school.
All made before ChatGPT existed :D.

---

## Projects

| Project | Description | Tech |
|---------|-------------|------|
| [remote-controlled-car](remote-controlled-car/) | Arduino-powered RC car controlled via Bluetooth | Arduino, C++ |

| [lucky-bet](lucky-bet/) | Simulated gambling platform with multiple casino games | PHP, MySQL, JS, CSS |
| [robotic-arm](robotic-arm/) | Arduino-powered robotic arm | Arduino, C++ |

### [robotic-arm](robotic-arm/)

Arduino-powered robotic arm. Built entirely from whatever was lying around the house — Lego pieces, loose wires, a disassembled old controller, and a small motor with a propeller that pulls a wire to close the gripper. Resourcefulness at its finest.
